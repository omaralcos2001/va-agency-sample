# CANDACE Agency

A modern, fully responsive virtual assistant agency demo website. Static demo only — no backend, database, auth, or API calls.

Built with Next.js 14 (App Router), Tailwind CSS, and lucide-react icons.

## Pages

- `/` — Home (hero, features, services, industries, process, testimonials, stats, CTA)
- `/services` — All 15 service lines with tasks, benefits, and industries
- `/about` — Mission, vision, story, values, team
- `/success-stories` — Case studies
- `/apply` — VA careers page with positions, application form (UI only), FAQ
- `/contact` — Contact form (UI only) and office info

## Run locally

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Deploy on Vercel

Push this repo to GitHub, then import it at https://vercel.com/new — Vercel auto-detects Next.js. No configuration needed.
